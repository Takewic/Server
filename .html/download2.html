Como fazer o Download?



Tem duas maneiras para fazer o Download do programa: A primeira sera nescessario ter o git instalado no seu computador, tendo o git instalado você abrira o terminal e digitara o seguinte comando:

"git clone https://github.com/Takewic/FastTester "

A segunda você apenas precisara acessar o github e fazer o download do repositorio "FastTester" depois disso, abra o terminal, entre na aba Download e digite "unzip FastTester-master.zip" , depois acesse o arquivo "FastTester-master" , e depois execulte o "install.sh".

Pronto o programa foi instalado agora basta apenas execultar o programa digitando "takewic".
